## About the mod

The aim of ModernMC is to add better biomes, more well-rounded mobs and mechanics and nicer shaders (the shaderpack is called MakeUp).

## Features

- Explore better biomes
- See nicer visuals
- Endless QoL updates
- Run it on a potato
