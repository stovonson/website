## About the mod

The goal of Harmony is to include as many mods as possible from the most popular optipacks (optimization modpacks), to create the best possible level of optimization.

## Features

- Best possible optimization
- Get (up to) 5x normal FPS
- Great for Potato PCs
- Run shaders with Iris
